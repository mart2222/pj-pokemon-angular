# Olá :cat:


## Projeto Pokémon em Angular 11

  Usado a api: https://pokeapi.co/api/v2

> Comando usados para iniciar o projeto:
> 
> ng new pokemon-djeison
> 
> npm i --save serve
> 
> npm i --save bootstrap
> 
> npm i --save ngx-bootstrap
> 
> npm i --save bootstrap-icons


## Intalação

`$ npm install`

---

## Iniciar

`$ npm start`

---

## Gerar arquivos

`$ npm run build`
